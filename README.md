## 🛠️ Troubleshooting common issues

If you encounter problems during the installation or usage, check these common fixes first.

Plugin does not appear in your workstation:
Most software requires a plugin rescan. Go to your software preferences and find the Plugin Manager or Plugin Path settings. Click the Rescan button to allow the host to find the files in the directory where you performed the installation.

Installer reports a security warning:
Modern versions of Windows sometimes flag unknown software. If a blue box appears, click More Info, then click Run Anyway. This confirms you trust the source.

Audio sounds thin or distorted when using the plugin:
Verify your input levels. If your original audio signal is already peaking, add a gain stage before the plugin to turn it down. This gives the plugin enough room to process the dynamics of the track without causing signal clipping.

Version conflicts:
Ensure you do not have older copies of the same plugin installed in different folders. Mixing old and new versions of plugins causes instability. Remove old versions from your VST or VST3 folders before running the new installer.
